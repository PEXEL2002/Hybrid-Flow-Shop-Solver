# Data Folder

This folder contains input files with HFS-SDST problem instances.

## Files

- **default.json** - Default problem instance used for testing
- **input_*.json** - Instances automatically generated by the application (timestamp)

## JSON File Format

```json
{
  "algorithm": "tabu",
  "num_stages": 2,
  "num_jobs": 3,
  "machines_per_stage": [1, 2],
  "learning_coeff": 0.15,
  "learning_stages": "10",
  "processing_times": [
    [[10], [8, 8]],
    [[12], [9, 9]],
    [[8], [7, 7]]
  ],
  "setup_times": [
    [
      [[[0], [0, 0]], [[2], [1, 1]], [[3], [2, 2]]],
      [[[2], [1, 1]], [[0], [0, 0]], [[4], [3, 3]]],
      [[[3], [2, 2]], [[4], [3, 3]], [[0], [0, 0]]]
    ],
    [
      [[[0], [0, 0]], [[2], [1, 1]], [[3], [2, 2]]],
      [[[2], [1, 1]], [[0], [0, 0]], [[4], [3, 3]]],
      [[[3], [2, 2]], [[4], [3, 3]], [[0], [0, 0]]]
    ]
  ]
}
```

## Note

`input_*.json` files are automatically cleaned up and should not be committed to the repository.
